Projects

Observed

In this google chrome extention, a statement is made on data tracking and online privacy by covering web page content with eyes that track the mouse. technologies used in this project include the p5js library, JavaScript, css, and the google chrome extension API.

Extending the Body

This web project uses machine learning and data visualization to trace a person's path through time by leaving a trace of their past position on camera with a colorful skeleton whose colors change based on how old they are. Technologies used in this project include the p5js library, the ML5 library, and JavaScript.

Try it here!

pocket adventure app

This mobile app prototype for android acts as a choose your own adventure style game using swipe-based controls. Adventures the user goes on can also be stored and viewed later. Technologies used in this project include Dart, Flutter, and dart packages including Carousel Slider and Google Fonts.
